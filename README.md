# web_project_homeland

Proyecto web estático responsive creado durante el Sprint 5. Contiene una página principal y varios estilos organizados en bloques CSS reutilizables, imágenes y recursos de vendor.

**Estado:** Terminado

**Características principales:**
- **Responsive:** diseño adaptable a móviles y escritorio.
- **Estructura modular de estilos:** archivos CSS por bloque en la carpeta `blocks/`.
- **Recursos incluidos:** fuentes y normalización en `vendor/`.

**Estructura del proyecto (resumen):**
- `index.html` — entrada principal del sitio.
- `index.css` — estilos globales (en la raíz).
- `blocks/` — CSS por componente (ej.: `intro.css`, `lead.css`, `photo.css`, etc.).
- `images/` — imágenes del proyecto.
- `pages/` — (carpeta para páginas adicionales si aplica).
- `vendor/` — recursos de terceros (`normalize.css`, `fonts.css`, `fonts/`).
